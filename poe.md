# キーワード
## ジョブ
CoCの職業のようなもの。この世界では技能を使わないので、職業、興味ポイントは存在しない。  
能力値は職業ごとに決まっており、自身で割り振りすることはない。  
ただし、技能の値を参照することはある。〈回避〉など。  
## アイテム
CoCの装備のようなもの。この世界ではアイテムを装備することによって様々な効果を得られる。  
アイテムごとにPrefixとSuffixが付いており、これによってHPやMP、〈回避〉や〈ブロック〉が変動する。  
特定のPrefixとSuffixが付いているアイテムも存在する。このアイテムはユニークアイテムと呼び、PrefixとSuffixを削除することも増やすことも出来ない。
```
装備箇所
武器：右手、左手  
防具：頭、胴体、手袋、足  
装飾：指輪（右手）、指輪（左手）アミュレット  
```
## スキル
CoCの呪文のようなもの。スキルジェムとサポートジェムが存在し、スキルジェムは4つまで付けることが出来る。サポートジェムはスキルジェムに2つまで付けることが出来る。  
ただし、1つだけ5つ付けることが出来る。  
```
トルネードショット
├投射物強化   
├アタック元素ダメージ  
├低体温化  
├インスピレーション  
├クリティカルダメージ増加  
```
