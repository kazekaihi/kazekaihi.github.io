# 仕様説明
## 職業
  職業ごとにレベルが上った時の恩恵が違う。1度決めたら変更できない。技能の最大値は80である。    
## レベル
自身のレベル。アイテムを装備するのに必要になる。経験値を貯める方式ではなく、同じレベルのマップをクリアすることによって上がる。初期1。  
## アイテム
CoCの装備のようなもの。この世界ではアイテムを装備することによって様々な効果を得られる。装備条件があり、満たしてなければ装備できない。  
種別は  
武器は〈近接武器〉〈遠距離武器〉〈魔法〉に分類される。〈片手〉と〈両手〉があり〈片手〉ならもう1つ〈片手〉を着けられる。武器を2つ持っていたとしても技能を使えるのは、片方だけである。  
防具は〈服〉〈軽装〉〈重装〉に分類される。  
指輪とアミュレットは記載がなければ全員付けることが出来る。  
つまりキャラシートに書く時のテンプレートは以下のようになる。  
```
装備中
武器　〇〇
防具　〇〇
指輪　〇〇
アミュレット　〇〇
```
## 実績
特定の行動や業績を成した者に与えられる称号のようなもの。ダサいものもある。「ふりがな」の所に1つだけ書いておける。書いてあるものだけが、効果を発揮する。  
## 能力値
基本的にCoCと変わらないが、APPとEDUは基本12固定である。  
APPは使わないから、EDUは職業ポイントが240になるようにする調整のためである。  
## 追加技能
職業やアイテムによって追加で技能を得られるようになる可能性がある。ビルドを考えるうえで重要になるので読み込んでおくと良い。  
## マップ
遺跡。異界。周回コンテンツ。基本的にここを周回する。死んでも戻るだけ。何度でも周回して稼ぎを上げよう。敵を倒した後、1d100を行う。特定の数値以下なら、ドロップが手に入る。
## PvP
そういう場所がある。行く必要はない。アイテムを賭けて複数人で争う事ができる。別に行く必要はない。負けた場合賭けたアイテムは相手に渡る。本当に行く必要はない。
# 職業
CoCの職業に書く。1度決めたら変更することは出来ない。職業ポイント計算式はEDU×20。  
## 勇者
全てができる代わりに全てが何かに劣っている職業。
```
レベルアップ時の恩恵：APP、SIZ、EDU以外の能力値にランダムに+2
取得可能な技能：〈近接武器〉〈遠距離武器〉〈魔法〉〈服〉〈軽装〉〈重装〉〈回避〉
使用可能な武器種：全て
初期装備：全ての職業の初期装備の中から3つ選ぶ。
```
## 戦士
体力が多く、前線を張るタンクのような立ち位置。代わりに頭が悪いので魔法は使えない職業。
```
レベルアップ時の恩恵：STR、CON+1
取得可能な技能：〈近接武器〉〈重装〉〈回避〉
使用可能な武器種：〈近接武器〉
初期装備：鉄のソード　ヘビーアーマー
```
## 盗賊
体力が低い代わりに高い行動力を持つ。もちろん魔法は使えない職業。
```
レベルアップ時の恩恵：DEX+2
取得可能な技能：〈近接武器〉〈遠距離武器〉〈軽装〉〈回避〉
使用可能な武器種：〈近接武器〉〈遠距離武器〉
初期装備：鉄のダガーor鉄の弓　レザーアーマー
```
## 魔法使い
体力が低く武器も使えない。代わりに魔法に優れている職業。
```
レベルアップ時の恩恵：POW、INT+1
取得可能な技能：〈魔法〉〈服〉〈回避〉
使用可能な武器種：〈魔法〉
初期装備：鉄のワンドor鉄のセプター　ローブ
```

# 追加技能
近接武器
```
〈援護防御〉：敵の攻撃を自身が受けることが出来る。この時、〈回避〉は出来ない。
〈受け流し〉：敵の攻撃を受け流す。受けるダメージを半分にする。
```
遠距離武器
```
〈ヘッドショット〉：〈遠距離武器〉と併用する。与えるダメージを武器のダメージと同じ値追加する。
```
魔法
```
〈火球〉：コスト4MP。4d6の魔法ダメージ。
〈HP回復〉：コスト4MP。対象一人の耐久力2d6回復。
〈MP回復〉：コスト1MP。自身のPOWと同じ値MPを回復。
```
# 実績
## 雑魚
```
取得条件：なし
効果：装甲+2
```
## 伝説
```
取得条件：職業が勇者
効果：レベルが上がる時、追加で+1上がる。
```
## 伝令
```
取得条件：グラッセを討伐
効果：凍結を付与されない。
```
# マップ
## 砂浜
**推奨**：Lv. 1~5  
**エネミー**：グラッセ  
**ドロップ**：10%：ゴアドリル 5%:ライフスプリッグ 1%皆をつなぐもの 
### グラッセ
**耐久力**：50×探索者の中の最大レベル（最大5）  
**攻撃**：〈魔法〉1d3人に1d8ダメージ。25%の確率で対象は凍結する。凍結：1d3ラウンドスタン。  
**装甲**：なし
