# キーワードと仕様説明
## ジョブ
CoCの職業のようなもの。この世界では技能を使わないので、職業、興味ポイントは存在しない。  
能力値は職業ごとに決まっており、自身で割り振りすることはない。  
ただし、技能の値を参照することはある。〈回避〉など。
ジョブごとにレベルが上った時の恩恵が違う。クソデカパッシブツリーはない。    
## レベル
自身のレベル。装備やスキルを付けるのに必要になる。経験値を貯める方式ではなく、同じレベルのマップをクリアすることによって上がる。最大100。敵がドロップする装備の要求レベルは敵のレベルと同じになる。  
## アイテム
CoCの装備のようなもの。この世界ではアイテムを装備することによって様々な効果を得られる。装備条件があり、満たしてなければ装備できない。  
アイテムごとにPrefixとSuffixが付いており、これによってHPやMP、〈回避〉や〈ブロック〉、能力値など多彩な数値が変動する。  
特定のPrefixとSuffixが付いているアイテムも存在する。このアイテムは**ユニークアイテム**と呼び、PrefixとSuffixを削除することも増やすことも出来ない。
```
装備箇所
武器：右手、左手  
防具：頭、胴体、手袋、足、ベルト  
装飾：指輪（右手）、指輪（左手）アミュレット  
```
## スキル
CoCの呪文のようなもの。スキルジェムとサポートジェムが存在し、スキルジェムは4つまで付けることが出来る。サポートジェムはスキルジェムに2つまで付けることが出来る。  
ただし、1つだけ5つ付けることが出来る。  
例  
```
トルネードショット
├投射物強化   
├アタック元素ダメージ  
├低体温化  
├インスピレーション  
├クリティカルダメージ増加  
```
## 戦闘
この世界の戦闘は敵から行動する。敵の攻撃が終わった後、プレイヤーの攻撃が出来る。（アタック、スペル）攻撃後、敵が生きていたならまた敵の攻撃と言った感じである。  
マナが0になっても、ライフが1になっても気絶はしない。どちらかが死ぬまで殴り合う、そんな戦闘である。  
## ダメージ
攻撃時に与えるダメージの計算式。  
```
ダメージ =
(基礎ダメージ + 追加ダメージ)
* 増加ダメージ
* (クリティカルダメージボーナス * クリティカルヒット率)（DOTは無い）
* アタックスピード/キャストスピード（DOTは無い）
```
## 被ダメージ
被弾時に受けるダメージの計算式。"~のダメージとして受ける" の修整、ダメージ軽減、ダメージ量修整の順で行われる。1万の火ダメージを受ける時、火耐性75%、受ける火ダメージ‐100の場合  
```
（10000*0.25）‐100=2400
```
となる  
## クリティカル率
1d100で5以下を出す方式ではない。計算方法は  
```
クリティカル率 = (基本クリティカル率 + 加算修整) * (1 + 乗算修整) 
```
最小確率は5%。スキルによってはクリティカルが発生しないものもある。  
## 耐性
ダメージ軽減。この世界のダメージの属性は5種類。元素耐性とは火、冷気、雷のことを指す。初期最大値75%、最大値90%。  
```
物理　初期耐性0%
火　初期耐性0%、レベル50で‐30%、レベル80で-60%
冷気　初期耐性0%、レベル50で‐30%、レベル80で-60%
雷　初期耐性0%、レベル50で‐30%、レベル80で-60%
混沌　初期耐性0%、レベル50で‐30%、レベル80で-60%
```
混沌属性はエナジーシールドを無視してダメージを与える。  
## ライフ
初期50。レベルが上がるごとに8上がる。STRが10ごとに+5上がる。
計算式は
```
 (50 + (レベル*8) + (STR*0.5) + "最大ライフ実数値") * (100% + "最大ライフ倍率")
```
## マナ
初期40。レベルが上がるごとに4上がる。INTが10ごとに+5上がる。
計算式は  
```
 (40 + (レベル*4) + (INT*0.5) + "最大マナ実数値") * (100% + "最大マナ倍率")
```
ラウンドごとに最大マナの2%のマナを回復する。
##　エナジーシールド
仮HPのようなもの。混沌属性以外のダメージをライフよりも先にエナジーシールドで受ける。装備や職業でしか増えない。  
## アーマー
物理ダメージ耐性のようなもの。ダメージ軽減。アーマーの数値と受けるダメージを比較して軽減率が決まる。
```
アーマーがダメージより5倍多い　50%軽減
アーマーがダメージより3倍多い　25%軽減
アーマーがダメージと等倍　10%軽減
アーマーがダメージより少ない　軽減なし
```  
## 回避
アタックに対する防御レイヤー。ヒット前判定。スペルとDOTは防げない。回避の数値と受けるダメージを比較して回避率が決まる。回避に成功した次のターン、回避は半分になる。これは失敗するまで半分になり続ける。  
```
回避がダメージより3倍多い　100%で回避する
回避がダメージより2倍多い　75%で回避する
回避がダメージと等倍　50%で回避する
回避がダメージより少ない　回避できない
```
## ブロック
攻撃を無効化する防御レイヤー。ヒット前判定。アタックをブロックする〈ブロック〉とスペルをブロックする〈スペルブロック〉がある。DOTは防げない。  
## リザーブ
マナを一定割合、もしくは実数値スキルを解除するまで減らす。スキルによってはライフをリザーブするかも知れない。  
## マップ
CoCのシナリオのようなもの。敵と戦闘をし、勝てば報酬がもらえる。負ければ戻って来る。入るために条件がある場所もある。  
## その他
チャージ：チャージは存在する間その個数分バフを得るステータス。最大個数は3  
エンデュランスチャージ：チャージ1つにつき物理ダメージ耐性+4%、属性耐性+4%  
フレンジーチャージ：チャージ1つにつきダメージ+4%  
パワーチャージ：チャージ1つにつきクリティカル率40%増加  
状態異常：様々なものがある。出血、毒、凍結、冷却、感電、燃焼。状態異常がつかない敵もいる。幸運なことに自身につかないようにも出来る。  
幸運：2回ロールし、ダメージであれば高い方を、〈回避〉や〈ブロック〉は成功した方を採用する。  
不運：2回ロールし、ダメージであれば低い方を、〈回避〉や〈ブロック〉は失敗した方を採用する。  
スペル抑制率：スペルダメージを半分にする確率。ダメージ軽減。  

# ジョブ
## マローダー
所謂戦士、筋肉モリモリマッチョマンの変態。レベルが上がるごとにライフ+4、STR+5、INTとDEX+2。レベル20でライフ+10%、レベル30で元素耐性+20、レベル40で被ダメージ耐性10%、レベル50で上位職業の一つを選択。レベル80で元素耐性最大値+5%。  
```
ジャガーノート　
被ヒット時にエンデュランスチャージを最大数得る。エンデュランスチャージの最大数+1、エンデュランスチャージの効果が2倍
```
```
バーサーカー
受けるダメージ+10%、ダメージ+40%、物理ダメージ+100%。
```
```
チーフテン
火耐性の半分を冷気と雷耐性に追加する。火耐性の最大値が全ての耐性の最大値に適用される。火耐性の最大値+5%
```
## レンジャー
所謂弓兵、回避マン。レベルが上がるごとにエナジーシールド+5、DEX+5、STRとINTが+2。レベル20でエナジーシールド+10%、レベル30で混沌耐性+20、レベル40でダメージ+10%、レベル50で上位職業の一つを選択。レベル80で混沌耐性+30%。
